Notre mission

BinWatch est un prototype orienté data et analytics permettant d’améliorer la gestion des déchets urbains grâce à l’analyse d’images et à la structuration des observations terrain.

L’objectif est d’anticiper les débordements de poubelles, de cartographier les zones à risque et d’aider à une prise de décision plus efficace via un pipeline complet : upload, classification, stockage, historique et visualisation.

À propos de BinWatch

BinWatch est une plateforme web qui permet de détecter l’état des poubelles publiques à partir de simples photos.

Le projet combine interface web, API Node.js, service Python de vision par ordinateur et base PostgreSQL pour produire des données exploitables dans une logique de monitoring environnemental.

Fonctionnalités principales

Upload d'images

Téléversez une photo pour analyser automatiquement l’état d’une poubelle.

Détection intelligente

Le système classe les poubelles comme pleines ou vides à partir de règles analytiques.

Carte et dashboard

Visualisez les zones critiques, l’historique et des indicateurs utiles au suivi.
